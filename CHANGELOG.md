## 0.0.2

Initial release — a Flutter port of the Sileo React toast library.

- A single morphing, gooey "liquid pill" notification driven by spring physics.
- Six intents (`success`, `error`, `warning`, `info`, `loading`, `action`) and an imperative `sileo.*` API (incl.
  `future`, `dismiss`, `clear`).
- `Toaster` host: six positions, offsets, light/dark/system theming, autopilot, hover-to-pause, swipe-to-dismiss,
  reduced-motion.
- Zero third-party dependencies.
